CEF Engine — How It Works

Signal generation → allocation modeling → daily trade execution. Designed for $60M+ SMA portfolios with institutional liquidity constraints.

Fair Value Engine (Gridiron Proprietary)

Proprietary buy/sell discount levels set per fund based on historical analysis. Generates a 0-100 FV Score based on distance from fair value buy target. Blended with other signal layers using configurable weights.

Layer 1 — Z-Score Mean Reversion

Current discount vs historical average in std devs. Weighted blend of 63/126/252 day lookbacks.

Layer 2 — Peer Relative Value

Cross-sectional discount rank vs sector peers.

Layer 3 — Momentum Overlay

Prevents buying falling knives. Disc narrowing + price trend + volume.

Layer 4 — Seasonal + Events

Calendar effects (Jan effect, tax-loss selling) and distribution changes.
